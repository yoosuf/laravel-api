# Support

## Getting Help

For usage questions and troubleshooting:

- Open a GitHub Discussion (recommended for Q&A)
- Open a GitHub Issue (for reproducible bugs)
- Read existing docs in docs/

## Before Opening an Issue

Please include:

- Laravel version
- PHP version
- Package version
- Minimal reproduction steps
- Expected behavior
- Actual behavior
- Logs or stack trace

## Issue Types

- Bug reports: for confirmed or suspected defects
- Feature requests: for enhancements and roadmap ideas
- Security reports: use SECURITY.md private process only

## Maintainer Response

Best effort support is provided for supported versions.
Response times may vary based on complexity and maintainer availability.
